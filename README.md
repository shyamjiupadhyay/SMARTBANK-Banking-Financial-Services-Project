# SMARTBANK-Banking-Financial-Services-Project

💻 SmartBank – Frontend Banking System
📌 Overview

SmartBank is a frontend-based banking and financial services application that simulates real-world banking operations. It allows users to perform activities like registration, login, balance checking, money transfer, and transaction tracking.

The project is built using core web technologies and does not require any backend. All data is stored using browser localStorage.

🚀 Features
🔐 User Registration & Login
💰 Balance Display
➕ Add Money
🔁 Transfer Money
📜 Transaction History
🔍 Search Transactions
📊 Spending Chart (Chart.js)
🌙 Dark / Light Mode
🛠️ Tech Stack
HTML
CSS
JavaScript
Chart.js
LocalStorage

smartbank/

⚙️ How It Works
User registers or logs in
Data is stored in localStorage
Dashboard loads user details
User performs transactions
Data updates instantly on UI
Graph shows transaction summary
📊 Graph System

The application uses Chart.js to visualize financial data such as spending and income. It helps users understand their transaction patterns in a graphical format.

🎯 Purpose

This project is developed for educational purposes to understand how banking systems work and how frontend technologies can be used to simulate real-world applications.

⚠️ Limitations
No backend or database
Data stored locally (not secure)
No real transactions
